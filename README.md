## My-CV
You can view the live version of the project here.

## Description
"My-CV" is a personal CV website designed to showcase professional experience, skills, and contact information in a visually appealing format. The project is deployed on Netlify, making it easily accessible and shareable.

## Technologies Used
HTML - For the content structure of the website
CSS - For styling and layout
JavaScript - For interactivity and dynamic elements
React.js - For creating a component-based and responsive user interface
Netlify - For deployment and automatic updates
